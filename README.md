# Fangjiangyi.github.io
Jiangyi Fang, Home page.

## Blog posts

Write blog posts as Markdown files under `_posts/`.

Use this filename format:

```text
YYYY-MM-DD-post-title.md
```

Each post should start with front matter:

```yaml
---
layout: post
title: "Post Title"
date: 2026-06-17
categories: notes
tags: [research, notes]
---
```

The blog index is available at `/blog/`.
