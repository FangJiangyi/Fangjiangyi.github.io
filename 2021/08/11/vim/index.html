模式分类

  1. command mode
  2. insert mode

这种模式分类可以使我们利用键盘完成全部任务，不必分心去动鼠标。
有关模式切换，现下只了解两点：

  • 无论当前处于何种mode输入Esc总可以退回到command mode。
  • press i enter insert mode

Lesson 1

cursor move

h(left)   j(down)   k(up)   l(right)

start vim from the shell prompt type

command: vim FILENAME <ENTER>

delete the charactor at the cursor type

x

To insert or append text type

  • i   type inserted text:insert before the cursor
  • A  type inserted text <ESC>: append after the line

Lesson 2

DELETION COMMANDS

dw to delete a word when we are in the command mode.

d$ to delete to the end of the linewhen we are in the command mode.

command grammar
operator [count] motion

[x] x means the element of the set of parameters that can be put onto that location.

operator:

Now we just learn d operator for delete.

motion:

  • w - until the start or the next word, EXCLUDING its first character.
  • e - to the end of the current word, INCLUDING the last character.
  • $ - to the end of line, INCLUDING the last character.
    For example, d2w means delete 2 words from the cursor.

MOVE TO THE START

type 0

UNDO COMMANDS

To undo previous actions, type:    u  (lowercase u)
To undo all the changes on a line, type:    U  (uppercase U)
To undo the undo’s, type:    CTRL-R

THE PUT COMMAND

function:Type p to put previously deleted text after the cursor.

THE REPLACE COMMAND

function type rx to replace the character at the cursor with x.(under the condition that it is in the command mode)

THE CHANGE OPERATOR

grammar:
operator [number] motion
c$(from the line where cursor is placed)
c2e
…

Lesson 4

GOTO COMMAND

CTRL-G:show information about the current file and current line’s number.
G go to the last line of the file
gg go to the first line of the file
[number] G go to the numberth line of the file

THE SEARCH COMMAND

type / followed by a phrase to search for the phrase
n:next(in one direction)
N:Next(in the other direction)
?:search for a phrase in the backward direction.
CTRL-o go back to where you came from
CTRL-i goes forward

THE MATCH COMMAND

% is used to match ‘{}’,’[]’ and ‘()’

THE　SUBSTITUTE COMMAND

decide what function these commands below means:

s/old/new(/g)
#,#s/old/new/g
%s/old/new/g
%s/old/new/gc

EXECUTE AN EXTERNAL COMMAND

type :! followed by an external command to execute that command

WRITE FILES

:w FILENAME

MORE ON WRITE FILES

v motion :w FILENAME

RETRIEVING AND MERGING files

:r FILENAME: to insert the contents of a file——retrieves disk file FILENAME and puts it below the cursor position

Lesson 6

  1. Type o to open a line BELOW　the cursor and start Insert mode.
  2. Type O to open a line ABOVE the cursor.
  3. Type a to insert text AFTER the cursor.
  4. Type A to insert text after the end of the line.
  5. The y(for yank) operator yanks text, p puts (pastes) it.
  6. Typing a capital R enters Replace mode until <ESC> is pressed.
  7. Typing “:set xxx” sets the option “xxx”. Some options are:
  • ‘ic’ ‘ignorecase
  • ‘is’ ‘incsearch’
  • ‘hls’ ‘hlsearch’

what are these three options means?
