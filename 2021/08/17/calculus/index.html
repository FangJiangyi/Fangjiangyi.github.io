导数与偏导数

导数

导数定义：$f’(x_0)=\lim \limits_{\Delta x\rightarrow0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$

高阶导数采用递归定义：$f ^{(n)}(x)=\lim \limits_{\Delta x \to 0}\frac{f^{(n-1)}\ \ (x_{0}+ \Delta x)-f^{(n-1)}\ \ (x_{0})}{\Delta x}$

积法则证明

\begin{align}
\lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0}+\Delta x)-u(x_{0})v(x_{0})}{\Delta x} &= \lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0}+\Delta x)-u(x_{0}+\Delta x)v(x_{0})+u(x_{0}+\Delta x)v(x_{0})-u(x_{0})v(x_{0})}{\Delta x}\\
&= \lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0}+\Delta x)-u(x_{0}+\Delta x)v(x_0)}{\Delta x}+\lim\limits_{\Delta x\to 0}\frac{u(x_{0}+\Delta x)v(x_{0})-u(x_{0})v(x_{0})}{\Delta x}\\
&= u(x_{0})v’(x_{0})+u’(x_{0})v(x_{0})
\end{align}

商法则证明

\begin{align}
\lim\limits_{\Delta x \to 0} \frac{\frac{u (x_{0}+\Delta x)}{v(x_{0}+\Delta x)}-\frac{u (x_{0})}{v(x_{0})}}{\Delta x}&=\lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0})-v(x_{0}+\Delta x)u (x_{0})}{\Delta x v(x_{0}+\Delta x)v (x_{0})}\\
&= \lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0})-u(x_{0})v (x_{0})}{\Delta x v(x_{0}+\Delta x)v (x_{0})}+\lim\limits_{\Delta x \to 0}\frac{u (x_{0})v(x_{0})-u (x_{0})v(x_{0}+\Delta x)}{\Delta xv (x _{0}+\Delta x)v(x_{0})}\\
(\frac{u(x)}{v (x)})’\bigg|_{x=x_{0}}&= \frac{u’(x_{0})v (x_{0})-u (x_{0})v’(x_{0})}{v_{0}^2}
\end{align}

复合函数求导法则证明

\begin{align}
\lim\limits_{\Delta x \to 0}\frac{f (g (x_{0}+\Delta x))-f (g (x_{0}))}{\Delta x} &=\lim\limits _{\Delta x \to 0}\frac{f (g (x _{0}+\Delta x))-f (g (x _{0}))}{g (x _{0}+\Delta x)-g (x _{0})}\lim\limits _{\Delta x \to 0}\frac{g (x _{0}+\Delta x)-g (x _{0})}{\Delta x} \\
&=\lim\limits _{\Delta x \to 0}f’(g (x _{0}))g’(x _{0})
\end{align}

小结1

以上的证明仅仅要求$f,g,在x_{0}处可导对于复合函数求导法则（f在g(x_{0})处可导）$从这当中回应我们的导数运算法则是不依赖于导函数的存在性的。也就是f,g在某一点的可导性，g在该点的非零性等可以推出以上运算后的函数在某点的导数的存在性以及其计算公式。同样从复合函数求导公式可以了解到：$f’和f$是一对对应关系（函数和其导函数可建立对应关系）只要f确定（不管以哪一种形式确定（只要符合函数的定义）——这里引申是f形式存在可以不依赖于自变量）则其导函数f’就确定了。因此f’(2x)是对于f’这一函数代入2x计算。

中值定理与微分应用

由于不给出罗尔定理的证明这里可以说一下大致思路：
由于连续函数则区间必有最值，如果两个最值均在端点处取到，则得出区间恒为定值，可得出导数=0；如果有一个不在端点处取到，则成为极值，可考察极值性质。

定义极值点，$若x_0使得存在领域B _{r}(x _{0}) ,对任意的x \in B _{r}(x _{0}),f(x) \le或\ge f(x _{0}) 则称x_0为极值点$
费马引理：极值点导数=0；反证法

罗尔定理

$函数f(x)在区间[a,b]上连续，在区间(a,b)上可导，且f(a)=f(b)。则有\eta \in (a,b),满足f’(\eta)=0;$
罗尔定理不给出证明

拉格朗日中值定理

$函数f(x)在区间[a,b]上连续，在区间(a,b)上可导。则有\eta \in (a,b),满足f’(\eta)=\frac{f(b)-f(a)}{b-a};$
证明

柯西中值定理

$函数f(x),g(x)在区间[a,b]上连续，在区间(a,b)上可导。则有\eta \in (a,b),满足\frac{f’(\eta)}{g’(\eta)}=\frac{f(b)-f(a)}{g(b)-g(a)};$
证明

总结

从几何的角度来理解证明的过程：罗尔中值定理是基本，拉格朗日中值定理是对坐标进行了线性变换的罗尔中值定理，柯西中值定理是参数化曲线的罗尔中值定理。
