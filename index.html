传感器

绪论

传感器的定义

传感器是将各种非电量（包括物理量、化学量、生物量等）按一定规律转换成便于处理和传输的另一种物理量（一般为电量）的装置

传感器的发展趋向

三“新”

  1. 新材料的开发应用
  2. 新工艺技术的应用
  3. 新效应的发现

Remark:
新材料意味着本质上的区别（能够实现数量级上的突破）？
新工艺技术代表着成本降低、性能优越
新效应的发现——光电效应的发现引领了光电传感器的发明，同样，新的效应，可以构建不同物理量之间的关系，其中新的传感器就可能利用这样的关系。
五“化”

  1. 固态化
  2. 集成化和多功能化
  3. 图像化
  4. 智能化
  5. 网络化

固态化
集成化——提高稳定性（分立元件和集成元件的对比）自带输出处理以及可视化模块等
多功能化——类似于软件，我们希望我们做的是通用编程和通用算法可扩展性强
图像化——可视化问题是永久的且是重要的
智能化——能够针对不同情况做出一些不同的调整（但基本仅限于有限状态机，并不能说具备了智能）
网络化——响应实时数据，上传云端

静态校准曲线

在标准工作状态下，利用一定精度等级的校准设备，对传感器进行往复循环测试，即可得到输出-输入数据。将这些数据列成表格，再画出各被测量值（正行程和反行程）对应输出平均值的连线，即为传感器的静态校准曲线。

拟合直线的求解

  1. 过零拟合
  2. 过零旋转拟合
  3. 端点连线拟合
  4. 端点平移拟合
  5. 最小二乘法

其中端点连线拟合又称为端基法，实现方法：
把传感器校准数据的零点输出平均值$a _{0}$和满量程输出的平均值 $b _{0} $连成的直线 $a _{0}b _{0} $作为传感器特性的拟合直线。

线性度（非线性误差）

定义

在规定条件下，传感器校准曲线与拟合直线间最大偏差与满量程输出值的百分比称为线性度

性质

非线性误差是以一定的拟合直线或理想直线为基准直线算出来的——基准直线不同，所得线性度也不同

Remark

谈论线性度和非线性误差时，必须同时说明所依据的基准直线。不能笼统而谈。

单枢电阻存在的问题：转矩小、波动大、有0转矩点（存在位置动力不足）。

想法1：
我们可以想到，当增加更多圈电枢框时，即使有些电枢框处在零转矩位置，但由于随着通电电枢框数量的增加，可以认为处在零转矩位置的电枢提供的力矩越来越微不足道，此时可认为电动机提供的转矩越来越平稳。

想法2：
观察力矩随空间分布的曲线，对于增加电枢个数的操作可以认为是同一电枢曲线经过位移（处在不同位置）的曲线的叠加，叠加过后我们会发现，两者没有零转矩点而且下降的幅度变小。

想法3：
可以认为不同位置的电枢电阻贡献的转矩是一个曲线上的不同的点，这些电阻贡献转矩的累加可以被认为是积分过程（差了倍数），积分使得取值更加平均，可以减缓变化过快的地方（起到低通滤波的作用）

导数

导数定义：$f’(x_0)=\lim \limits_{\Delta x\rightarrow0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$

高阶导数采用递归定义：$f ^{(n)}(x)=\lim \limits_{\Delta x \to 0}\frac{f^{(n-1)}\ \ (x_{0}+ \Delta x)-f^{(n-1)}\ \ (x_{0})}{\Delta x}$

积法则证明

\begin{align}
\lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0}+\Delta x)-u(x_{0})v(x_{0})}{\Delta x} &= \lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0}+\Delta x)-u(x_{0}+\Delta x)v(x_{0})+u(x_{0}+\Delta x)v(x_{0})-u(x_{0})v(x_{0})}{\Delta x}\\
&= \lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0}+\Delta x)-u(x_{0}+\Delta x)v(x_0)}{\Delta x}+\lim\limits_{\Delta x\to 0}\frac{u(x_{0}+\Delta x)v(x_{0})-u(x_{0})v(x_{0})}{\Delta x}\\
&= u(x_{0})v’(x_{0})+u’(x_{0})v(x_{0})
\end{align}

商法则证明

\begin{align}
\lim\limits_{\Delta x \to 0} \frac{\frac{u (x_{0}+\Delta x)}{v(x_{0}+\Delta x)}-\frac{u (x_{0})}{v(x_{0})}}{\Delta x}&=\lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0})-v(x_{0}+\Delta x)u (x_{0})}{\Delta x v(x_{0}+\Delta x)v (x_{0})}\\
&= \lim\limits_{\Delta x \to 0}\frac{u(x_{0}+\Delta x)v(x_{0})-u(x_{0})v (x_{0})}{\Delta x v(x_{0}+\Delta x)v (x_{0})}+\lim\limits_{\Delta x \to 0}\frac{u (x_{0})v(x_{0})-u (x_{0})v(x_{0}+\Delta x)}{\Delta xv (x _{0}+\Delta x)v(x_{0})}\\
(\frac{u(x)}{v (x)})’\bigg|_{x=x_{0}}&= \frac{u’(x_{0})v (x_{0})-u (x_{0})v’(x_{0})}{v_{0}^2}
\end{align}

复合函数求导法则证明

\begin{align}
\lim\limits_{\Delta x \to 0}\frac{f (g (x_{0}+\Delta x))-f (g (x_{0}))}{\Delta x} &=\lim\limits _{\Delta x \to 0}\frac{f (g (x _{0}+\Delta x))-f (g (x _{0}))}{g (x _{0}+\Delta x)-g (x _{0})}\lim\limits _{\Delta x \to 0}\frac{g (x _{0}+\Delta x)-g (x _{0})}{\Delta x} \\
&=\lim\limits _{\Delta x \to 0}f’(g (x _{0}))g’(x _{0})
\end{align}

小结1

以上的证明仅仅要求$f,g,在x_{0}处可导对于复合函数求导法则（f在g(x_{0})处可导）$从这当中回应我们的导数运算法则是不依赖于导函数的存在性的。也就是f,g在某一点的可导性，g在该点的非零性等可以推出以上运算后的函数在某点的导数的存在性以及其计算公式。同样从复合函数求导公式可以了解到：$f’和f$是一对对应关系（函数和其导函数可建立对应关系）只要f确定（不管以哪一种形式确定（只要符合函数的定义）——这里引申是f形式存在可以不依赖于自变量）则其导函数f’就确定了。因此f’(2x)是对于f’这一函数代入2x计算。

罗尔定理

$函数f(x)在区间[a,b]上连续，在区间(a,b)上可导，且f(a)=f(b)。则有\eta \in (a,b),满足f’(\eta)=0;$
罗尔定理不给出证明

拉格朗日中值定理

$函数f(x)在区间[a,b]上连续，在区间(a,b)上可导。则有\eta \in (a,b),满足f’(\eta)=\frac{f(b)-f(a)}{b-a};$
证明

柯西中值定理

$函数f(x),g(x)在区间[a,b]上连续，在区间(a,b)上可导。则有\eta \in (a,b),满足\frac{f’(\eta)}{g’(\eta)}=\frac{f(b)-f(a)}{g(b)-g(a)};$
证明

总结

从几何的角度来理解证明的过程：罗尔中值定理是基本，拉格朗日中值定理是对坐标进行了线性变换的罗尔中值定理，柯西中值定理是参数化曲线的罗尔中值定理。

$1第一章布尔逻辑

这章我最想说明两个问题：

  1. abstraction(抽象)：讨论是什么。这个是什么，指的是功能上的描述，或者叫下定义，架构在此abstraction基础上的事物只需要利用这个abstraction而不需要关注具体实现的细节。举例：尽管实现与非门有各种各样的工艺、材料、方法，但是这是(implementation)层次的具体细节，我们封闭不管。我们所关心的内容——无论是如何实现的与非门，其逻辑功能是不变的——也正是我们关心的。因此，我们抽象出与非门这个概念，凡是能实现这个逻辑功能的东西都和与非门等价。后续本门课所有的内容都在利用与非门的abstraction。

  2. implementation(实现)：讨论如何。也就是利用上一级的abstraction使得某一abstraction（人脑中的概念）变成事实上的内容。举例：如何具体制造出与非门。需要半导体物理以及电子方面的知识。

Remark:

有关abstraction和implementation需要注意的事情：抽象应该是唯一的；实现是多样的。抽象应该具有完备性：也就是在某一层进行实现时，本层提供的抽象应该完全能够实现所有功能。实现应该具备短视性：我们仅用与目标层相邻底层提供的抽象就应该足够实现目标层的抽象。抽象不仅仅是一个方法这么简单，其隐含的是一种哲学思想，抽象在某种程度上，与哲学中利用概念的方法是类似的，但是不同的是，我们要做的是利用抽象上升到更高层抽象；

在第一章所创造的各种硬件芯片中，唯一值得注意的是多路数据选择器和多路数据分配器在数据传输中的应用。以及数据选择器可以作为控制逻辑出现。

Lesson 2

DELETION COMMANDS

dw to delete a word when we are in the command mode.

d$ to delete to the end of the linewhen we are in the command mode.

command grammar
operator [count] motion

[x] x means the element of the set of parameters that can be put onto that location.

operator:

Now we just learn d operator for delete.

motion:

  • w - until the start or the next word, EXCLUDING its first character.
  • e - to the end of the current word, INCLUDING the last character.
  • $ - to the end of line, INCLUDING the last character.
    For example, d2w means delete 2 words from the cursor.

To undo the undo’s, type:    CTRL-R
